## O que foi criado

Um script Python que automatiza **todo** o seu processo:

1. **Abre links** do Sheets (Alt+Enter no primeiro, ↓ + Alt+Enter nos próximos)
2. **Espera 25s** para as extensões Tampermonkey fazerem o trabalho
3. **Processa cada aba** com Insert — usando uma técnica de "espiar" a próxima aba (Ctrl+Tab → verificar título → Ctrl+Shift+Tab) para saber se é a última
4. **Aperta End** na última aba (gera CSV) e volta ao Sheets
5. **Repete** automaticamente para o próximo produto
6. **Parável com ESC** a qualquer momento

## Como rodar

```
python automacao.py
```

Você tem 5 segundos de contagem regressiva para focar no navegador com o Sheets aberto.

## Configurações

Se precisar ajustar tempos (por exemplo, se 25s não for suficiente), as constantes ficam no topo do arquivo — basta alterar `WAIT_EXTENSIONS`, `WAIT_AFTER_KEY`, etc.
